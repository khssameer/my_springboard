Recommendation systems
-----------------------

A recommender system is a simple algorithm whose aim is to provide the most relevant information to a user by discovering patterns in a dataset. 
The algorithm rates the items and shows the user the items that they would rate highly.

Building Book Recommendation system. :

  - Use a Data set from http://www2.informatik.uni-freiburg.de/~cziegler/BX/ .
  - Data Wrangling / cleaning.
  - Exporatory Data Analysis.
  - Build various types of recommender models.
  - Below are recommendation methods I would like to use and come up with predictions.
      1. Popularity-based.
      2. Popularity-based.[localized to country and state]
      3. Collaborative Filtering Based recommendations.
        - Memory Based
        - Model Based
      4. Content Based recommendations.
